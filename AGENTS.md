# Encore Bio Labs Project Rules

Build a premium biotech website.

Design style:
Apple × Function Health × Hims × Moderna × Vercel.

Use:
- deep navy
- pearl white
- pharmaceutical silver
- soft teal accents
- glassmorphism
- premium typography
- subtle molecule backgrounds
- floating vial animation

Rules:
- Mobile first
- Clean spacing
- Reusable components
- Product data must live in src/data/products.ts
- Product images go in src/assets/images/products
- Logo goes in src/assets/images/logo
- Each product appears once
- Product variants live inside each product object
- Do not duplicate product cards for different strengths

Main sections:
- Navbar
- Hero
- Category grid
- Featured products
- Quality section
- How it works
- FAQ
- Footer
